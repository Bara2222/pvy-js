1: Co je JavaScript a k jakým účelům se používá?

Doplň vhodné odpovědi a využij k tomu nečíslovaného seznamu.

Použij v textu aspoň 3 internetové odkazy na zajímavé příklady využití JS!

  • JavaScript je vysokoúrovňový, interpretovaný, programovací jazyk.
  • Používá se pro vývoj dynamických a interaktivních webových stránek i aplikací.
  • Také k vytváření interaktivních map nebo zobrazování animovaných grafů.

2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. Camelcase (kombinace velkých, malých písmen = CamelCase).
  2. Příkazy se oddělují středníkem.
  3. Escape sekvence (pokud je nějaký speciální znak, píše se u toho obrácené lomítko).
  4. Řetězce se uzavírají do uvozovek.
  5. Programové sekvence se uzavírají do složených závorek {}.

Doplňte další možnosti ve formě kódu.

4: Jak je možné v JavaScriptu deklarovat proměnné?

Uveďte na příkladech různé datové typy, které je možné využívat v JS.

Doplňte chybějící kód/text podle vzoru úvodních dvou řádků!

let x = 0; // s přiřazením hodnoty
let x; // bez přiřazení hodnoty
let x = null; // přiřazuje prázdnou nebo neznámou hodnotu
let x = 8; // celé číslo (typ number)
let y = 5.36; // reálné číslo (typ number)
let z = 0.2e1; // zápis desetinného čísla pomocí exponentu
let txt = 'Znakový řetězec'; // typ string
let prestupny = false // logický typ boolean

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

Přiřazovací

= (přiřazení), *=, -=, /=, += (přičtení), ++ (přičtení 1), --(odečtení 1)

Logické

== (rovnost), != (nerovnost), <,<=,>=,> ( aritmetické srovnání), && (a zároveň), || (nebo), ! (negace), ?:(ternární operátor)

Porovnávací

>= (větší nebo rovno), <= (menší nebo rovno), > (větší), < (menší)

Bitové

<< (bitový posun vlevo, vrací dvojnásobek hodnoty), >> (bitový posun vpravo, vrací celočíselný podíl), ^ (bitové XOR), | (bitové OR), & (bitové AND)

Typové

typeof (typ proměnné), instanceof (kontrola typu objektu za běhu)

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* Vypíše upozornění do popup okna */
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
let password = prompt('Zadej heslo');

confirm()

/* Vyskočí popup okno s přesměrováním na stránku seznamu */
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování:
Většina prohlížečů popup okna blokuje.
